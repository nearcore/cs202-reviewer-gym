# Bug Fix Lab Patch

Adds the second IntelliJ Quest mode: Bug Fix Lab.

Counts:
- 25 base bug-fix problems shared by Level 1 and Level 2
- 25 extra Level 1 guided bug-fix problems
- 25 extra Level 2 solo bug-fix problems
- 25 Level 3 Code Review / Refactor problems
- 100 total Bug Fix Lab items

Behavior:
- Level 1 has one hidden optional clue
- Level 2 has no clue
- Level 3 is a separate code review/refactor class of problem
- Official review appears only after the user clicks “I attempted it — show official review”
- Website includes reflection boxes for what was fixed, why it was wrong, and the concept involved
- Each item includes an IntelliJ file path and a matching Java practice file
