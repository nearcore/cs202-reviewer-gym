# Exceptions and Files Practice

## Topic
try/catch, NumberFormatException, NullPointerException, Scanner, and file input.

## Goal
Handle a predictable problem without crashing the entire program.

## Instructions
Run the solution. It creates no files; pass a file path as an optional command-line argument to read it.

## Starter Code
`starter/FileStarter.java`

## Expected Output
The solution reports a parsed number or a useful error message.

## Questions Before Running
1. Which line is risky?
2. What exception might it throw?
3. What can you do after catch?

## Reflection
1. Where could a null value originate?
2. What topic does this connect to? String conversion.
