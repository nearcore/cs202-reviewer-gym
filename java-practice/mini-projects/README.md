# CS202 Mini Projects

Each file is a compact runnable reference version. Read its class-level comment, predict one output line, run it, then add one feature.

| Project | File | Practice focus |
| --- | --- | --- |
| Grade Analyzer | `GradeAnalyzer.java` | arrays, loops, methods, return values, averages |
| Student Directory | `StudentDirectory.java` | classes, objects, constructors, ArrayList, toString |
| Shape Drawing | `ShapeDrawing.java` | Graphics, methods, constants, colors |
| Animal Inheritance Lab | `AnimalInheritanceLab.java` | inheritance, super, overriding, polymorphism |
| KeyListener Move Square | `../08-interfaces-keylistener/MoveSquareGame.java` | KeyListener, fields, repaint, focus |
| File Word Counter | `FileWordCounter.java` | Scanner, files, split, exceptions |
| Records vs Classes Demo | `RecordsVsClassesDemo.java` | record, class, canonical behavior, equality |
| Final Mixed OOP Challenge | `FinalOopChallenge.java` | interface, ArrayList, inheritance, equals, toString |

For the GUI projects, close the window to stop the program. `FileWordCounter` needs a text-file path:

```bash
javac FileWordCounter.java
java FileWordCounter ../README.md
```
