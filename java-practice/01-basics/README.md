# Java Basics Practice

## Topic
Variables, types, naming, assignment/comparison, casting, strings, and `var`.

## Goal
Explain the type of each value and distinguish `=`, `==`, and `.equals()`.

## Instructions
Trace `trace/TraceBasics.java`, fix `debug/BrokenComparison.java`, then complete the starter.

## Starter Code
`starter/BasicsStarter.java`

## Expected Output
The solution prints a converted number and an integer-division result.

## Questions Before Running
1. Which expression uses integer division?
2. What type does `var name = "Ana"` infer?
3. Why is `=` not a comparison?

## Reflection
1. What conversion needs a method rather than a cast?
2. What topic does this connect to? Loops and conditions use every one of these basics.
