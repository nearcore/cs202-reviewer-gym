# Bug Fix Lab

Open these files in IntelliJ when the website asks for them.
Some files intentionally fail to compile or run until you fix them.

- `base/`: reusable bugs for Guided Fix and Solo Fix
- `level1/`: extra guided bug-fix tasks
- `level2/`: extra solo bug-fix tasks
- `level3/`: code review/refactor challenges; code may run or fail, and may contain design, logic, style, concept, or multiple issues

Use the website reflection boxes to explain what you fixed and why.
