# Java Practice Lab

Each numbered folder has the same learning pattern:

- `starter/` — code to complete.
- `solution/` — runnable reference implementation.
- `trace/` — predict output before running.
- `debug/` — intentionally broken code; identify the issue before fixing it.
- `build/` — a short coding challenge.

## Run a File

Compile and run from the directory containing the file:

```bash
javac FileName.java
java FileName
```

Java files do not use packages so each folder can be practiced independently. Do not compile every folder at once: several exercises intentionally reuse familiar class names such as `Student` or `Dog`.

`mini-projects/` contains runnable reference versions of all eight required projects.
