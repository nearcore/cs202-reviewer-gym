# Arrays and ArrayLists Practice

## Topic
Indexes, loops, 2D arrays, fixed-size arrays, and growable ArrayLists.

## Goal
Read and change collection data without going past the valid indexes.

## Instructions
Trace the index examples, repair the last-index bug, then complete the starter.

## Starter Code
`starter/ArrayStarter.java`

## Expected Output
The solution prints the last array item and a changed list item.

## Questions Before Running
1. What is the last valid index?
2. Which ArrayList method replaces an item?
3. How many rows are in a 2D array?

## Reflection
1. Did you use `length` or `size()` correctly?
2. What topic does this connect to? Loops.
