# Classes and Objects Practice

## Topic
Fields, methods, objects, references, access modifiers, and `this`.

## Goal
Create a small object with private data and controlled behavior.

## Instructions
Trace two references to one object, fix field shadowing, then complete the starter.

## Starter Code
`starter/StudentStarter.java`

## Expected Output
The solution prints a student after a field update.

## Questions Before Running
1. Which variables are fields?
2. Which reference points to the object?
3. Why use `this.name`?

## Reflection
1. What changes when two references share one object?
2. What topic does this connect to? equals and constructors.
