# Static, Final, and Memory Practice

## Topic
Class fields, instance fields, constants, final methods/classes, and reachability.

## Goal
Tell whether state belongs to one object or every object, and use final precisely.

## Instructions
Trace the shared counter, fix the constant name, then complete the starter.

## Starter Code
`starter/CounterStarter.java`

## Expected Output
The solution prints a shared count of two and a constant limit.

## Questions Before Running
1. How many `count` fields exist?
2. Can a final reference’s object still change?
3. When is an object eligible for garbage collection?

## Reflection
1. Why can Java still have memory leaks?
2. What topic does this connect to? Constructors and object state.
