# Equality and Object Text Practice

## Topic
Assignment, reference equality, logical equality, and `toString`.

## Goal
Override `equals(Object)` and `toString()` correctly.

## Instructions
Trace String comparison, fix the bad equals header, then complete the starter class.

## Starter Code
`starter/BookStarter.java`

## Expected Output
The solution prints true for logically equal books and a useful description.

## Questions Before Running
1. What does `==` compare for objects?
2. Why must equals accept `Object`?
3. When does Java call toString automatically?

## Reflection
1. What field values define equality here?
2. What topic does this connect to? Records generate equals and toString.
