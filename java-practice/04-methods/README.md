# Methods Practice

## Topic
Headers, parameters, return values, method calls, and overloading.

## Goal
Write a small method that returns a value instead of only printing it.

## Instructions
Trace nested calls, fix a missing return, then complete the starter methods.

## Starter Code
`starter/MethodBasics.java`

## Expected Output
The solution prints the larger number and whether a value is even.

## Questions Before Running
1. What is the return type?
2. Which values are actual parameters?
3. What happens if a non-void method reaches the end without return?

## Reflection
1. Did you confuse printing with returning?
2. What topic does this connect to? Object methods.
