# Inheritance and Polymorphism Practice

## Topic
`extends`, `super`, overriding, abstract classes, parent references, and `instanceof`.

## Goal
Trace dynamic dispatch and model an actual is-a relationship.

## Instructions
Run the animal solution, fix the bad override, then extend the starter with another animal.

## Starter Code
`starter/AnimalStarter.java`

## Expected Output
The solution prints each child class’s overridden sound.

## Questions Before Running
1. What is the reference type? What is the object type?
2. Which `speak` method runs?
3. Is the relationship is-a or has-a?

## Reflection
1. When is `instanceof` useful?
2. What topic does this connect to? Interfaces.
