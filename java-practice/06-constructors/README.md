# Constructors Practice

## Topic
Default, overloaded, copy, and constructor chaining.

## Goal
Initialize objects into a valid starting state.

## Instructions
Trace `this(...)`, fix the constructor declaration, then create copies in the starter.

## Starter Code
`starter/ConstructorStarter.java`

## Expected Output
The solution prints a copied student.

## Questions Before Running
1. Does a constructor have a return type?
2. When does it run?
3. What does `this(...)` call?

## Reflection
1. What makes a copy constructor different?
2. What topic does this connect to? `super(...)` in inheritance.
