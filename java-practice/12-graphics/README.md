# Graphics Practice

## Topic
Swing `paintComponent`, colors, coordinates, and drawing methods.

## Goal
Draw a simple picture without putting drawing code in the wrong place.

## Instructions
Run the solution. Close its window to end the program. Add one shape in the starter.

## Starter Code
`starter/GraphicsStarter.java`

## Expected Output
The solution opens a window with a house and sun.

## Questions Before Running
1. Which calls fill versus outline shapes?
2. What do width and height mean in fillRect?
3. Why call super.paintComponent(g)?

## Reflection
1. What changes if repaint is not called after state changes?
2. What topic does this connect to? KeyListener.
