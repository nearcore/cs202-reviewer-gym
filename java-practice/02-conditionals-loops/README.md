# Conditions and Loops Practice

## Topic
Boolean logic, if, for, while, switch, modulo, break, and continue.

## Goal
Trace a loop without guessing and choose safe loop boundaries.

## Instructions
Predict the trace output, repair the off-by-one bug, then finish the starter loop.

## Starter Code
`starter/LoopStarter.java`

## Expected Output
The solution prints even numbers and their total.

## Questions Before Running
1. When does the loop condition become false?
2. Which variable changes every iteration?
3. What does `% 2` tell you?

## Reflection
1. What off-by-one error did you check for?
2. What topic does this connect to? Array indexing.
