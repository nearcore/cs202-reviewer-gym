# Website-only Bug Fix Lab Patch

Adds Bug Fix Lab without IntelliJ dependency.

Counts:
- 25 base bug-fix problems shared by Level 1 and Level 2
- 25 extra Level 1 guided bug-fix problems
- 25 extra Level 2 solo bug-fix problems
- 25 Level 3 Code Review / Refactor problems
- 100 total Bug Fix Lab items

Behavior:
- Level 1 has one hidden optional clue
- Level 2 has no clue
- Level 3 is a separate code review/refactor class of problem
- Users inspect the code snippet inside the website
- Users write what they would fix and why
- Official review appears only after the attempt
- Progress records whether the user understood the fix or needs review

No IntelliJ workflow or generated Java practice files are required for this patch.
